# UpcomingPaymentsComponent

Steps to deploy:
1. Deploy the force-app/main/default/classes/ and force-app/main/default/lwc folders to your Sandbox for testing.
2. After deployment is complete, add the Upcoming Payments component to the Account lightning page layout
3. Test various use cases to ensure it is displaying as expected
4. Deploy to production and specify the tests to run. Choose only the UpcomingPaymentsServiceTest.cls test class
5. Place on your account page layout in production
